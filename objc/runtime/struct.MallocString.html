Methods

impl MallocString

unsafe fn new(ptr: *mut c_char) -> Option<MallocString>

Constructs a new MallocString for a malloc'd C string buffer. Returns None if the given pointer is null or the C string isn't UTF8. When this MallocString drops, the buffer will be free'd.

Unsafe because ptr must point to a valid, nul-terminated C string.
